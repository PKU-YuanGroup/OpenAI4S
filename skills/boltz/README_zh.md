# Boltz Skill

本目录保存 Boltz-2 蛋白质/核酸/配体 co-folding 与可选 affinity 输出的渐进披露 recipe。它帮助 Agent 选择输入、运行外部 package，并解读 mmCIF 与 confidence Artifact；它本身不是 Boltz 实现。

Package、weights、GPU 与可选 MSA 服务均须另行可用。Affinity/confidence 输出属于模型预测，不能证明实验结合或生物学有效性。
