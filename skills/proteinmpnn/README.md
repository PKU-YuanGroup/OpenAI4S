# ProteinMPNN Skill

This progressive-disclosure recipe covers inverse folding of a protein backbone with the external ProteinMPNN repository. It is an operational runbook, not a bundled model implementation.

Generated sequences require folding, functional, developability, and experimental validation. CPU may be sufficient for small runs, while actual runtime and GPU benefit depend on campaign size and environment.

## Direct files

| --- | --- |
| [`SKILL.md`](SKILL.md) | Explains repository setup, PDB/chain selection, fixed/tied positions, sampling-temperature syntax, batch JSONL helpers, output FASTA interpretation, resource choice, and validation workflow. |
