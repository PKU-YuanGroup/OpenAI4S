# LigandMPNN Skill

This progressive-disclosure recipe covers inverse folding with ligand, nucleic-acid, and metal context. It guides use of the external LigandMPNN repository and checkpoints; no executable or weights are bundled here.

Generated sequences and threaded structures are design candidates, not experimentally validated binders. Chain, fixed-residue, context-atom, and model-type choices must be checked against the actual input structure.

## Direct files

| File | Responsibility |
| --- | --- |
| [`SKILL.md`](SKILL.md) | Documents repository setup, model types, PDB/context handling, design/fixed chains and residues, sampling, batch outputs, threaded PDBs, ligand-aware caveats, and downstream validation. |
