# Chai-1 Skill

This directory is the progressive-disclosure recipe for Chai-1 all-atom co-folding of proteins, nucleic acids, and ligands. It guides an agent through the external `chai-lab` Python API and result interpretation; it does not ship the model.

Availability depends on the package, weights, GPU resources, and optionally an external MSA service. Predicted complex structures and confidence scores require independent validation.

## Direct files

| File | Responsibility |
| --- | --- |
| [`SKILL.md`](SKILL.md) | Describes multi-entity FASTA syntax, Python inference options, MSA/template choices, output candidates and scores, batching/performance, consensus use, caveats, and licensing. |
