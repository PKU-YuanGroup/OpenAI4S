# ESMFold2 Skill

This progressive-disclosure recipe covers the external Biohub ESMFold2/ESMFold2-Fast co-folding models and ESMC protein language models. No code, weights, Hugging Face access, or GPU environment is bundled here.

Model/backend/version availability must be checked in the active environment. Structure, mutation, contact, and interpretability outputs are computational results and should not be presented as experimentally validated.
