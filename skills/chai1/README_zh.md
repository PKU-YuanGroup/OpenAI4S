# Chai-1 Skill

本目录是 Chai-1 蛋白质、核酸与配体全原子 co-folding 的渐进披露 recipe。它指导 Agent 使用外部 `chai-lab` Python API 并解读结果；本目录不分发模型。

可用性取决于 package、weights、GPU 资源以及可选外部 MSA 服务。预测复合物结构与 confidence score 仍需独立验证。
