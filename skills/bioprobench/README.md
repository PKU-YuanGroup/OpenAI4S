# bioprobench

BioProBench — a multi-task benchmark for biological protocol understanding and reasoning. The recipe is in [`SKILL.MD`](SKILL.MD); the evaluation helpers it calls are in [`kernel.py`](kernel.py), the runnable scripts in [`Scripts/`](Scripts/), and the sample inputs in [`data/`](data/).

If you use this benchmark to evaluate models or agents, cite the original work — see the citation block at the top of `SKILL.MD`.
