# OpenFold3 Skill

This directory contains the progressive-disclosure recipe for the external OpenFold3 AlphaFold3-style complex predictor. It describes model setup and execution but does not include OpenFold3 code, databases, parameters, or a prepared environment.

Actual support for proteins, nucleic acids, ligands, templates, and accelerators depends on the installed upstream version and assets. Predicted structures and confidence values remain computational hypotheses.

## Direct files

| File | Responsibility |
| --- | --- |
| [`SKILL.md`](SKILL.md) | Guides installation/environment expectations, input JSON/entities, database/MSA/template preparation, inference options, output/confidence interpretation, resource constraints, and upstream licensing. |
