# Evo 2 Skill

This progressive-disclosure recipe covers DNA likelihood scoring, embeddings, generation, and variant comparison with Evo 2. It supplies operational guidance only; model code, checkpoints, and accelerator runtime are external.

GPU requirements, supported context lengths, checkpoint access, and generation quality must be verified in the active environment. Scores are model-derived and must not be treated as experimental variant effects.
