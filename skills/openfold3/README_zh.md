# OpenFold3 Skill

本目录保存外部 OpenFold3、AlphaFold3-style 复合物预测器的渐进披露 recipe。它说明模型 setup 与执行，但不包含 OpenFold3 代码、数据库、参数或已准备环境。

蛋白质、核酸、配体、template 与 accelerator 的实际支持取决于安装的上游版本和资产。预测结构与 confidence value 仍是计算假设。

## 直属文件

| 文件 | 职责 |
| --- | --- |
| [`SKILL.md`](SKILL.md) | 指导安装/环境预期、输入 JSON/entity、数据库/MSA/template 准备、inference option、输出/confidence 解读、资源限制与上游许可。 |
