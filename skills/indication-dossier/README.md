# Indication Dossier Skill

This progressive-disclosure Skill organizes a therapeutic-indication research dossier across population, epidemiology, biology, standard of care, regulatory precedent, and clinical trials. It is a research/writing recipe, not a medical recommendation or an automatically verified evidence database.

The agent must retrieve current authoritative sources, cite them precisely, preserve uncertainty/conflicts, and obey the anti-fabrication gates in the reference phases. No sidecar or live data source is bundled.

## Direct files

| File | Responsibility |
| --- | --- |
| [`SKILL.md`](SKILL.md) | Orchestrates the phased waypoint workflow, progress/resume behavior, source collection, synthesis, output files, and quality checks. |
