# SolubleMPNN Skill

This progressive-disclosure recipe explains the soluble-biased ProteinMPNN weights exposed by the external ProteinMPNN/LigandMPNN repositories. It is a model-selection and execution guide, not a bundled runtime.

The soluble prior can trade native recovery for surface-composition bias; generated sequences still require structure, expression, aggregation, function, and experimental validation.
