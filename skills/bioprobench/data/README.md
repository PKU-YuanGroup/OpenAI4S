# BioProBench Data

The full BioProBench testing dataset contains over 5,000 structured instances across five fine-grained tasks. To keep this repository lightweight, the full dataset is not hosted here.

**How to get the data:**
Please download the official dataset directly from Hugging Face: [https://huggingface.co/BioProBench](https://huggingface.co/BioProBench).

- [`sample_pqa_output.json`](sample_pqa_output.json) — a sample PQA output, for confirming the format parses.
