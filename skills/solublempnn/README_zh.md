# SolubleMPNN Skill

这个渐进披露 recipe 说明外部 ProteinMPNN/LigandMPNN 仓库暴露的 soluble-biased ProteinMPNN weights。它是模型选择与执行指南，不是捆绑 runtime。

Soluble prior 可能以 native recovery 换取表面组成 bias；生成序列仍需结构、表达、聚集、功能与实验验证。
