# ADMET Genetic References

These files are supplementary recipe material loaded on demand (for example through `host.skills.read`) so the main Skill remains progressively disclosed. They document contracts and design choices; they do not install dependencies or supply experimental validation.

## Direct files

| File | Responsibility |
| --- | --- |
| [`admet.md`](admet.md) | ADMET-AI installation/invocation, endpoint orientation and aggregation policy, runtime behavior, and troubleshooting. |
| [`data_contracts.md`](data_contracts.md) | Required molecule identity, candidate/generation columns, parent fields, canonical `operation_detail`, lineage invariants, and visualization expectations. |
| [`ga.md`](ga.md) | Starter GA structure, mutation/crossover options, chemistry filters, SA scoring, composite scoring, and diversity selection notes. |

## Direct subdirectories

None.
