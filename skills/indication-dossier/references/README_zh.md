# Indication Dossier 参考资料

这些按需文档把长证据工作流拆为可审计 phase 与持久 waypoint contract。它们指导检索和综合；任何一个都不是疾病 claim 的证据来源本身。

## 直属文件

| 文件 | 职责 |
| --- | --- |
| [`00-research-standards.md`](00-research-standards.md) | 跨 phase citation、anti-fabrication、tool-use、source-quality 与 confidence-calibration 规则。 |
| [`01-meta-initialization.md`](01-meta-initialization.md) | Phase 1：解析 indication identity/scope、alias、diagnostic frame、geography/time cutoff 与初始 metadata waypoint。 |
| [`02-epidemiology-research.md`](02-epidemiology-research.md) | Phase 2：diagnostic criteria、prevalence/incidence、demographics、risk factor、natural history 与 epidemiology source。 |
| [`03-biology-soc-research.md`](03-biology-soc-research.md) | Phase 3：disease mechanism、biomarker、approved/current care、guideline、outcome 与 unmet need。 |
| [`04-regulatory-trials-research.md`](04-regulatory-trials-research.md) | Phase 4：endpoint、regulatory precedent、landmark/failed trial、design、eligibility 与 source capture。 |
| [`05-synthesis.md`](05-synthesis.md) | Phase 5 dossier 结构与 action，把 waypoint 证据协调成带 citation 的报告和结构化输出。 |
| [`06-writing-style.md`](06-writing-style.md) | Inline citation placement、link specificity、冲突措辞、prose discipline 与最终来源检查。 |
| [`waypoint-schemas.md`](waypoint-schemas.md) | Progress、identity、epidemiology、biology/standard-of-care、regulatory/trial、evaluated-source、report 与 research-output waypoint 的 JSON/file contract。 |
